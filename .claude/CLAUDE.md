# 基本设置
- 语言：中文输出
- 图形：跟我交流的过程中如果需要绘制图形，请使用 `excalidraw`工具绘制图形并展示
- 记忆优先：减少文件读取，变更时同步到memory

# 任务执行
- 任务拆分：分析依赖关系
- 并行：无依赖子任务用对应agent并行执行
- 串行：有依赖子任务按顺序执行
- 可选择适合的SKILL

# 文档规范
- 命名：{两位序号}_{中文名称}.md（序号从01自增）
- 路径：docs/{目录序号}_{主题}/{子目录序号}_{子主题}/ ，目录名称使用中文
- 序号规则：已存在文件时从最大序号递增

# 文件操作
- 使用filesystem工具
- 重要配置变更同步到memory
- 标准化路径和命名

# 绘图工具
- 讨论方案时：使用`Excalidraw`工具绘制图形，并保存成 .excalidraw格式的文件
- 设计界面原型时：使用`figma`工具绘制UI原型
