# 语言偏好
- 使用中文输出回答和文档

# 记忆偏好
- 优先使用记忆，减少文件读取
- 如果刷新了文档或代码，请同步刷新到记忆中
- 执行`/init`命令时，同时刷新到记忆中

# 知识偏好
- 对于你不确定的知识，优先使用搜索工具搜索并归纳信息，以补充到知识库:
  - 对于API或SDK，可以使用`context7`获取信息
  - 对于其他项，可以使用`Tavily`获取信息

# 工作协同
- 优先把任务拆分成子任务，并分析任务之间的依赖关系
- 对于没有依赖关系的子任务: 优先使用对应场景的agent分配子任务的agent，并行执行任务
- 对于有依赖关系的子任务: 优先使用对应场景的agent分配子任务的agent，按依赖关系串行执行
- 对于任务可以选择适合的SKILL执行

# 文档交付件
- 文档名称: {文档两位数序号}_{中文文档名称}.md , 对每个目录下的文档，文档两位数序号从01开始自增
- 文档路径：docs/{目录两位数序号}_{中文目录主题}/{子目录两位数序号}_{中文子目录主题}/ ，其中：
	- 子目录为可选    
    - 目录两位数序号从01开始自增
    - 对每个目录下的子目录，子目录两位数序号从01开始自增
    - 如果目录和子目录下已存在文件或目录，序号从当前已存在的序号的最大值开始递增

# 文件操作规范
- 使用filesystem工具进行所有文件读写操作
- 重要配置变更后同步更新到memory中
- 使用标准化的文件路径和命名规范
