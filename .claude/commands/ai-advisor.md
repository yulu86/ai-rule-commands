## 功能说明

专业的AI模型咨询工具，使用多个AI模型进行复杂问题分析，提供多角度的技术建议和决策支持，确保方案的科学性和可靠性。

## 执行步骤

### 1. 问题定义和需求分析
- 检查用户是否提供了问题描述或决策需求参数
- 如未提供参数，使用AskUserQuestion询问具体的问题背景和期望结果
- 使用Explore工具和filesystem了解项目背景和技术环境
- 使用sequential-thinking分解复杂问题，明确分析维度

### 2. Agent协调启动多模型分析
- **主分析启动**: 使用Task工具启动ai-model-advisor代理进行专业AI咨询
  - 基于参数 $ARGUMENTS 进行针对性的多模型分析
  - 使用multi-model-advisor获取多个AI模型的多样化观点
  - 提供综合性的技术建议和决策支持
- **深度技术分析**: 根据问题类型，启动相关领域专家agent
  - **代码实现问题**: 启动senior-code-reviewer验证技术方案可行性
  - **架构设计问题**: 启动相关架构专家进行架构层面分析
  - **性能优化需求**: 启动web-performance-analyst进行性能影响评估

### 3. 多模型并行分析
- **问题建模**: 将用户问题转换为适合AI模型分析的标准格式
- **多模型查询**: 使用multi-model-advisor查询多个AI模型
  - 为不同模型设置针对性的系统提示和分析角度
  - 获取多样化的解决方案、观点和建议
  - 确保分析结果的全面性和客观性

- **结果整合**: 收集和整理所有模型的分析结果
  - 对比不同模型的观点和建议
  - 识别共识点和差异点
  - 分析各建议的优势和适用场景

### 4. 专业验证和技术评估
- **技术可行性验证**: 使用context7获取相关技术文档和标准
  - 验证建议方案的技术可行性
  - 检查是否符合行业最佳实践
  - 分析实施的技术难度和资源需求

- **案例研究**: 使用tavily-mcp搜索相关案例和实践经验
  - 查找类似项目的成功案例
  - 分析失败案例的教训和原因
  - 了解行业的最新发展趋势

- **质量保证**: 使用ide进行代码质量检查（如涉及代码实现）
  - 验证代码方案的语法正确性
  - 检查代码质量和潜在问题
  - 确保代码符合规范要求

### 5. 综合分析和决策支持
- **方案对比分析**: 使用sequential-thinking进行深度分析
  - 从技术可行性、成本效益、风险可控性等多个维度评估方案
  - 建立决策矩阵和评估体系
  - 识别各方案的关键成功因素和潜在风险

- **多角度验证**: 通过不同模型和专家的观点交叉验证
  - 确保建议的客观性和科学性
  - 减少单一模型或单一专家的偏见
  - 提高决策的准确性和可靠性

### 6. 最佳建议制定
- **综合建议**: 基于所有分析结果制定最佳建议
  - 整合多模型分析的优势观点
  - 结合专业验证的结果
  - 考虑具体的项目约束和需求

- **实施指导**: 提供具体的实施路径和操作指南
  - 分阶段的实施计划
  - 关键节点和里程碑设置
  - 风险应对和问题解决方案

### 7. 知识沉淀和学习积累
- **分析记录**: 将分析过程和结果记录到memory中
- **模式提取**: 提取有价值的决策模式和分析方法
- **案例库建设**: 建立成功案例和最佳实践的知识库
- **持续学习**: 跟踪AI技术发展趋势，更新知识体系

## Agent协作机制

### 主分析流程
1. **ai-model-advisor** 主导整个多模型分析流程
2. 根据问题复杂程度，启动相应的专业领域专家
3. 整合多模型分析和专业验证的结果

### 协作场景
- **技术方案验证**: 与senior-code-reviewer协作验证代码实现可行性
- **架构方案评估**: 与相关架构专家协作进行架构层面分析
- **性能影响评估**: 与web-performance-analyst协作评估性能影响
- **复杂决策分析**: 通过multi-model-advisor获取多角度建议

## 使用示例

### AI技术选型咨询
```bash
/ai-advisor "为我的电商项目选择最合适的AI推荐引擎"
```
- 分析不同推荐引擎的优缺点
- 结合项目需求提供最佳建议
- 评估实施难度和成本

### 技术方案验证
```bash
/ai-advisor "验证这个微服务架构方案的可行性"
```
- 多模型分析架构方案的技术可行性
- 识别潜在风险和挑战
- 提供优化建议和替代方案

### 复杂问题决策支持
```bash
/ai-advisor "是否应该从React迁移到Vue.js？"
```
- 多角度分析技术迁移的利弊
- 评估迁移成本和收益
- 提供决策支持建议

### AI解决方案验证
```bash
/ai-advisor "验证这个AI聊天机器人的技术方案"
```
- 多模型验证技术方案的可行性
- 分析技术实现的关键挑战
- 提供优化和改进建议

## 特色功能

### 多模型智能分析
- 同时查询多个AI模型获取多样化观点
- 交叉验证不同模型的建议和方案
- 综合多方面意见制定最佳策略
- 减少单一模型偏见，提高决策准确性

### 专业领域协作
- 结合专业领域专家的深度分析
- 验证技术方案的可行性和实用性
- 考虑具体实施的技术约束和条件
- 确保建议的专业性和可操作性

### 决策支持体系
- 建立科学的决策评估体系
- 提供量化的决策依据和分析
- 考虑风险、成本、收益等多个维度
- 支持复杂技术决策的科学制定

## 输出内容

### 分析报告
- **问题分析**: 问题的详细分析和多维解读
- **多模型观点**: 各个AI模型的分析结果和建议
- **专业验证**: 技术可行性和专业角度的验证结果
- **综合评估**: 方案对比和综合评估分析

### 决策建议
- **最佳方案**: 基于所有分析得出的最佳建议
- **实施指导**: 具体的实施步骤和操作指南
- **风险评估**: 潜在风险识别和应对措施
- **成功指标**: 方案成功的量化指标和评估标准

### 知识沉淀
- **分析模式**: 有价值的分析方法和思维模式
- **最佳实践**: 领域内的最佳实践和经验总结
- **案例参考**: 相关的成功案例和失败教训
- **技术趋势**: 相关技术领域的发展趋势和前景

通过系统化的多模型分析和专业验证，为复杂技术决策提供科学、可靠的支持。
