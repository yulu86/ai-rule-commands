## 功能说明

智能分析并解释指定代码的功能、逻辑和实现原理，支持逐行解析和上下文关联说明。

## 执行步骤

1. **参数验证**：
   - 检查用户是否提供了代码位置或代码片段参数
   - 如未提供参数，提示用户输入需要解释的代码位置或直接粘贴代码

2. **代码获取与分析**：
   - 根据用户提供的参数 $ARGUMENTS 读取目标代码
   - 分析代码的语法结构、逻辑流程和依赖关系 use context7
   - 识别代码的核心功能和技术要点

3. **智能解释**：
   - **总体概述**：简明扼要地说明代码的主要功能和用途
   - **结构分析**：解释代码的组织结构和各部分的作用
   - **逐行解析**：对于复杂代码，提供逐行或逐代码段的详细解释
   - **上下文关联**：说明代码与项目其他部分的关联和影响

4. **输出优化**：
   - 使用清晰的结构化格式展示解释内容
   - 对关键概念和技术术语提供额外说明
   - 根据代码复杂度调整解释的详细程度
