## 功能说明

快速分析当前变更并生成简洁的 commit 消息，适合日常开发中的快速提交。

## 执行流程

1. **快速状态检查**：
   - 检查 Git 状态和变更文件
   - 识别主要变更类型

2. **智能分类**：
   - 根据文件扩展名和内容自动判断类型
   - 生成简洁的 commit 消息

3. **快速执行**：
   - 默认添加所有变更文件
   - 使用生成的消息提交
   - 提供执行结果

## 使用示例

```bash
# 自动分析并提交所有变更
/quick-commit

# 提供简短描述
/quick-commit "修复登录bug"
```

## 特点

- **快速便捷**: 一键完成分析和提交
- **智能识别**: 自动判断变更类型和范围
- **简洁高效**: 适合日常高频使用
- **安全可靠**: 包含基本的安全检查
