# 代码审查检查清单

## 🔴 高优先级问题

### 安全性检查
- [ ] **输入验证**
  - [ ] 用户输入是否进行充分验证
  - [ ] 是否存在SQL注入风险
  - [ ] 是否存在XSS攻击风险
  - [ ] 文件上传安全性检查
  - [ ] API参数验证和清理

- [ ] **权限控制**
  - [ ] 认证机制是否健全
  - [ ] 授权检查是否正确实施
  - [ ] 敏感操作是否有权限验证
  - [ ] 会话管理是否安全
  - [ ] 密码和令牌处理是否安全

- [ ] **数据保护**
  - [ ] 敏感数据是否加密存储
  - [ ] 传输过程中数据是否加密
  - [ ] 日志中是否包含敏感信息
  - [ ] 错误信息是否泄露敏感数据

- [ ] **依赖安全**
  - [ ] 第三方库是否存在已知漏洞
  - [ ] 依赖版本是否及时更新
  - [ ] 镜像和包源是否可信

### 稳定性和可靠性
- [ ] **错误处理**
  - [ ] 异常是否被适当捕获和处理
  - [ ] 错误恢复机制是否完善
  - [ ] 资源清理是否正确
  - [ ] 边界条件是否考虑
  - [ ] 空值检查是否充分

- [ ] **并发安全**
  - [ ] 线程安全问题
  - [ ] 竞态条件处理
  - [ ] 死锁风险检查
  - [ ] 原子操作使用是否正确
