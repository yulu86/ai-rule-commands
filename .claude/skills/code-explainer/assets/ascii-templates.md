# ASCII 可视化模板

## 流程图模板

### 线性流程
```
开始 → 输入 → 处理 → 输出 → 结束
```

### 条件分支
```
输入数据
    ↓
条件判断
    ├─ 真 → 执行A → 结果A
    └─ 假 → 执行B → 结果B
```

### 循环结构
```
初始化 ←┐
    ↓    │
判断条件    │
    ├─ 真 → 执行体
    └─ 假 → 结束
```

### 简单处理
```

```

### 多步骤处理
```
原始数据 → 清洗预处理 → 数据转换 → 业务逻辑 → 结果格式化 → 最终输出
```

```

    ├─ 路由A → 处理器A → 响应A
    ├─ 路由B → 处理器B → 响应B
    └─ 路由C → 处理器C → 响应C
```
