# Excalidraw 绘图最佳实践

## 设计原则

### 1. 清晰性优先
- **简洁明了**: 避免过度复杂的设计
- **重点突出**: 重要元素使用醒目的颜色和样式
- **层次分明**: 通过大小、颜色区分重要程度
- **易于理解**: 使用标准的图形符号和表示方式

### 2. 一致性原则
- **颜色统一**: 整个图表使用统一的色彩方案
- **样式一致**: 相同类型的元素使用相同的样式
- **字体规范**: 统一字体族和大小规范
- **布局规则**: 遵循统一的布局和间距规则

### 3. 可读性设计
- **适当间距**: 元素间保持足够的间距
- **合理比例**: 图形大小与内容成比例
- **对比明显**: 文本与背景有足够的对比度
- **对齐规范**: 使用网格对齐保持整齐

## 颜色使用指南

### 基础配色方案
```css
/* 系统主色调 */
--primary-color: #1971c2;      /* 深蓝色 - 主要元素 */
--secondary-color: #40c057;    /* 绿色 - 成功状态 */
--warning-color: #f59f00;      /* 橙色 - 警告状态 */  
--danger-color: #fa5252;       /* 红色 - 错误状态 */
--neutral-color: #868e96;      /* 灰色 - 中性元素 */

/* 背景色 */
--bg-primary: #ffffff;         /* 白色 - 主背景 */
--bg-secondary: #f8f9fa;       /* 浅灰 - 次要背景 */
--bg-highlight: #fff3bf;       /* 浅黄 - 高亮背景 */
--bg-info: #d0ebff;            /* 浅蓝 - 信息背景 */
```

### 功能颜色映射
- **开始/结束**: 绿色 (#51cf66) / 红色 (#ff6b6b)
- **处理步骤**: 蓝色 (#339af0) 
- **决策点**: 橙色 (#ff922b)
- **数据存储**: 紫色 (#9775fa)
- **连接线**: 黑色 (#000000) / 深灰 (#495057)
- **注释说明**: 灰色 (#868e96)

### 颜色使用禁忌
- 避免使用超过5种主要颜色
- 不要使用相近的颜色区分重要元素
- 避免纯色搭配，适当使用柔和色调
- 确保颜色对比度符合WCAG标准

## 布局最佳实践

### 间距规范
```javascript
// 标准间距系统
spacing: {
  xs: 8,    // 最小间距 - 元素内部
  sm: 16,   // 小间距 - 紧密元素
  md: 32,   // 中等间距 - 一般元素  
  lg: 48,   // 大间距 - 主要分组
  xl: 64    // 最大间距 - 主要区域
}
```

### 对齐原则
1. **水平对齐**: 左对齐、右对齐、居中对齐
2. **垂直对齐**: 顶部对齐、底部对齐、居中对齐
3. **网格对齐**: 使用8px或16px网格系统
4. **视觉对齐**: 考虑视觉重心的对齐

### 层次结构
```
1. 画布边界
2. 主要区域分组
3. 子模块分组  
4. 具体元素
5. 连接线
6. 文本标注
```

## 文字规范

### 字体选择
```css
/* 推荐字体族 */
font-family-primary: "Virgil", "Segoe UI", sans-serif;    /* 主要文字 */
font-family-mono: "Cascadia Code", "Consolas", monospace; /* 代码文字 */
font-family-decorative: "Comic Sans MS", cursive;         /* 装饰文字 */
```

### 字体大小规范
```css
font-sizes: {
  xs: 12,    /* 注释、说明 */
  sm: 14,    /* 次要文字 */
  base: 16,  /* 正文文字 */
  lg: 20,    /* 小标题 */
  xl: 24,    /* 标题 */
  "2xl": 32, /* 主标题 */
  "3xl": 40  /* 大标题 */
}
```

### 文字使用原则
- **简洁表达**: 使用简短、精炼的表达
- **统一格式**: 相同类型内容使用相同格式
- **可读优先**: 确保文字大小便于阅读
- **避免重叠**: 文字不要与其他元素重叠

## 特定场景优化

### 流程图设计
1. **方向统一**: 整个流程图保持一致的方向
2. **节点间距**: 水平间距80-120px，垂直间距60-100px
3. **连接线**: 使用直线，避免过度弯曲
4. **决策标注**: 决策分支要有明确的条件标注

### 架构图设计
1. **层次分明**: 清晰展示不同架构层次
2. **模块区分**: 使用不同颜色区分不同模块
3. **接口标识**: 明确标注模块间的接口
4. **数据流向**: 用箭头清晰标识数据流向

### 思维导图设计
1. **中心突出**: 中心主题要醒目
2. **分支平衡**: 各分支保持视觉平衡
3. **层级递进**: 逐级缩进，体现层次
4. **颜色编码**: 使用颜色区分不同分支

## 常见错误避免

### 布局错误
- ❌ 元素排列杂乱无章
- ❌ 间距大小不一致
- ❌ 对齐不规范
- ❌ 元素重叠遮挡

### 颜色错误  
- ❌ 颜色过多过杂
- ❌ 对比度不足
- ❌ 颜色含义模糊
- ❌ 色彩搭配不协调

### 文字错误
- ❌ 文字过小难读
- ❌ 字体不统一
- ❌ 表达冗长复杂
- ❌ 文字与其他元素重叠

### 内容错误
- ❌ 流程逻辑错误
- ❌ 连接线错误
- ❌ 缺少必要标注
- ❌ 图形符号不规范
