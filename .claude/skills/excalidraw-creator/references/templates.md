# 常用图形模板

## 流程图模板

### 1. 基本流程图
```
[开始] → [数据输入] → [处理] → [判断] → [输出] → [结束]
                      ↓        ↓
                   [错误处理] [其他分支]
```

**元素配置**:
- 开始/结束：椭圆，绿色/红色背景
- 处理步骤：矩形，白色背景
- 决策点：菱形，黄色背景
- 连接线：箭头，黑色实线

### 2. 用户注册流程
```
[开始] → [填写信息] → {信息完整?} → [验证邮箱] → [注册成功] → [结束]
             ↓              ↓         ↓
          [显示错误] ← [邮箱验证失败] [发送失败]
```

### 3. 订单处理流程
```
[下单] → [库存检查] → {有库存?} → [支付处理] → [发货] → [完成]
          ↓               ↓          ↓
       [缺货提醒] ← [支付失败] → [取消订单]
```

## 使用说明

### 布局原则
1. **从左到右**: 流程遵循自然阅读方向
2. **从上到下**: 层级关系清晰展示
3. **间距一致**: 元素间保持合理间距
4. **对齐整齐**: 使用网格对齐功能

### 颜色规范
- **主色调**: 蓝色系 (#339af0, #1971c2)
- **辅助色**: 绿色系 (#51cf66, #40c057)  
- **警告色**: 橙色系 (#ff922b, #f59f00)
- **错误色**: 红色系 (#ff6b6b, #fa5252)
- **中性色**: 灰色系 (#868e96, #495057)

### 文字规范
- **标题**: 20-32px，粗体
- **正文**: 14-18px，常规
- **注释**: 12-14px，斜体
